# autoreturn 
autoreturn like that of ruby. 

## installation

```
pip install git+https://github.com/ytbryan/autoreturn/
```

## usage

```
from autoreturn import autoreturn

@autoreturn
def hello():
  "hello world" 

print(hello())
>> hello world
```

## original story

autoreturn is created to remove these return values
